testing my ideas

Testing My Ideas is a site that I designed and built from scratch. The site is a personal journal of all the ideas the user has tested out in the past. From the successful to the failed attempts, each project is recorded for both accountability and learning opportunities.

This site was created using the Next.js framework. Since the users needs were focused on speed and flexibility, I thought it would be a good idea to develop a static site that utilised the latest technologies the web had to offer.
